The Basic Network Sniffer is a Python-based cybersecurity project that captures and analyzes live network packets using the Scapy library. This tool monitors network traffic in real time and displays essential packet information such as source and destination IP addresses, protocol type (TCP/UDP), and port numbers.

The project is designed for learning and educational purposes, helping beginners understand how data travels across networks and how packet inspection works at a fundamental level.
